发表于 2018-07-18 | 分类于 javascript | 阅读次数

基本的格式化

语句结尾

推荐在语句结尾使用分号

行的长度

推荐 80 个字符， 需要手动换行

换行

在运算符之后换行， 第二行1个缩进（假定缩进为4个字符）

空行

  1. 在方法之间
  2. 在方法中的局部变量和第一条语句之间
  3. 在多行或者单行注释之前
  4. 在方法内的逻辑片段之前插入空行

命名

小驼峰式命名（第一个为小写字母）

变量：命名前缀应当是名词： 如

函数：命名前缀应当是动词

常量：使用大写字母和下划线来命名，下划线用来分割单词

构造函数： 大驼峰式命名， 第一个字母为大写

直接量

javascript 中包含一些类型的原始值：字符串，数字，布尔值，null和undefined,以及对象直接量和数组直接量

字符串：推荐双引号包裹，多行字符串连接使用+

数字：小数不能省略整数部分

null: 当作对象的占位符, 空对象指针。

无论在什么情况下，都不应该把一个变量现实的设置为undefined, 可是同样的规则对null却不适用。
只要意在保存对象的变量在还没有真正保存对象，就应该明确地让该对象保存null值。

下列场景应当使用null

  1. 用来初始化一个变量，这个变量可能赋值为一个对象。
  2. 用来和一个已经初始化的变量比较，这个变量可以事业可以不是一个对象。
  3. 当函数的参数期望是对象时，用做参数传入。
  4. 当函数的返回值期望是对象是，用作返回值传入。

还有一些场景不应当使用null

  1. 不要使用null来检测是否传入了某个参数。
  2. 不要用null来检测一个未初始化的变量。

undefined: 那些没有被初始化的变量都有一个初始值，即undefined

块状语句

不论块状语句包含多行代码，还是单行代码，都应当使用花括号，注意花括号对齐方式

switch 语句

在没有默认行为且写了注释的情况下省略default

变量、函数和运算符

变量申明

将所有的var语句合并为一个语句，每个变量的初始化独占一行。没有初始值得变量应该在尾部。

函数申明

函数申明不应当在语句块之内

立即调用的函数

通过在圆括号实现函数的立即执行。

严格模式

局部使用

相等

由于javascript 具有强制类型转换机制，推荐使用 === 和 !==

eval()

在JavaScript中 eval()并不是唯一可以执行JavaScript字符串的函数，使用Function 构造函数亦可以做到这一点。setTimeout()和setInterval()也可以。

  1. 禁止给setTimeout()和setInterval()传入字符串参数。

  2. 严禁使用 Function
    只在别无他法时使用 eval()

原始包装类型 (不要使用原始包装类型来创建新对象)

javascript 有3种原始包装类型: String、Boolean和 Number

每种类型都代表全局作用域中的一个构造函数，并分别表示各自对应的原始值的对象。
原始包装类型的主要作用是让原始值具有对象般的行为

将配置数据从代码中分离出来

配置数据：

  1. URL
  2. 需要展现给用户的字符串
  3. 重复的值
  4. 设置
  5. 任何可能发生变更的值

将CSS从JavaScript中抽离

避免使用全局变量

全局变量带来的问题

  1. 明明冲突
  2. 代码的脆弱性

任何来自函数外部的数据都应当以参数形式传进来。

意外的全局变量

当你给一个未被var 语句声明过的变量赋值时， JavaScript就会自动创建一个全局变量。

name实际上是window的一个默认属性。window.name 属性经常用于框架（frame）和 iframe 的场景中，当点击连接时，可以通过指定打开连接的目标容器来控制其在特定的框架或选项卡中显示，不小心修改name会影响到站点的连接导航。

避免“空比较”

检测原始值

javascript中有5种原始类型： 字符串、数字、布尔值、null 和 undefined。

使用typeof 检测原始类型，语法：

typeof variable

typeof “124”; // “string”

检测引用值

引用值也称作对象（object）。在JavaScript中除了原始值之外的饿值都是引用值。有这几种内置的引用类型： Object、 Array、 Date 和 Error。

typeof 判断这些引用类型时，都会返回 “object”。

检测某个引用值的类型最好的方法是使用instanceof 运算符。instanceof 的基本语法是：

value instanceof constructor

检测函数 （Function构造函数）

检测函数最好的方法是使用typeof, 因为它可以跨帧(frame) 使用。

检测数组

Array.isArray(需要检测);

检测属性

判断属性是否存在的最好办法是使用in 运算符。in 运算符仅仅会简单地判断属性是否存在，而不会去读属性的值。
