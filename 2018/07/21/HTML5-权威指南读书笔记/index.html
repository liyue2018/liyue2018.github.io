HTML5 权威指南读书笔记（一）

发表于 2018-07-21 | 分类于 html5 | 阅读次数
字数统计 1,225

第一部分 开篇

HTML5新增元素

details, summary, section, nav, footer, header, figure, figcaption, address, aside, article, hgroup

html5 全局属性

  1. contenteditable 属性（新增）

让用户能够修改页面上的内容

  1. dir 属性

ltr(用于从左到右的文字)
rtl(用于从右到左的文字)

  1. spellcheck 属性

只用在用户可以编辑的元素上

  1. tabindex 属性

HTML页面上的键盘焦点可以通过按Tab建在各元素之间切换。
tabindex=”-1” 不会被选中。

  1. try…catch…finally

第二部分 HTML元素

设置相对于URL的解析基准

base 元素中的href属性

为页面定义网站标志

异步加载执行脚本的两种方式

  1. defer 延迟执行
    属性，只用于外部脚本文件。浏览器在遇到带有defer属性的script元素时，会将脚本的加载和执行推迟到HTML文档中所有的元素都已得到解析之后。
    (能够按照顺序执行)
  1. async 属性 异步加载脚本和执行脚本。

noscript 元素

用基本的文字元素标记内容

  1. s元素

习惯样式 s { text-decoration: line-through; }

  1. small 元素

表示小号字体内容，常用与免责声明和澄清声明。

习惯样式 small { font-size: smaller; }

  1. sub 和 sup 元素

分别表示下标和上标。

  1. wbr 元素（新增）

wbr 用来表示长度超过当前浏览器窗口的内容适合在此换行，只是建议。

  1. abbr 元素用来表示缩写，其title属性表示的是该缩写代表的完整词语。
  1. ruby, rt, rp 元素（新增）

注音符号

  1. mark 元素（新增）

突出显示文字

mark { background: yellow; color: black; }

  1. time 元素（新增）

自定义属性

必须以 data- 开头

表格

表单

  1. autocomplete 自动完成功能 使用时要慎重考虑
  2. autofocus 自动聚焦
  3. fieldset 对表单元素编组 （禁用）
  4. 使用表单外的元素 通过form元素的id 属性 和 form属性设置
  5. maxlength 用户能够输入的字符的最大数目，size 是文本框在屏幕上的大小。
  6. datalist 元素 （新增） 与 list option 一起用
  7. 表单编码类型为multipart/form-data 的时候才能上传文件。
  1. 用 button 元素提交表单

submit
reset 重置

  1. 由于浏览器对input元素的处理方式向来都比较一致，更倾向于用其生成按钮。

  2. input type 还可以为 color range file image

表单验证

  1. required
  2. 确保输入值位于某个范围内
  1. 确保输入值于指定模式匹配

pattern 属性可以用来确保输入值与一个正则表达式匹配。

嵌入内容

  1. 推荐在img标签里指定 width height, 让浏览器能够在图像尚未载入时预留位置，预防抖动。
  1. iframe

使用 Document 对象

  1. document.URL 获取当前文档的URL
  2. window.location.href 或者 document.location.href 获取当前文档的URL
