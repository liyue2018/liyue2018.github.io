Vue.js

vue.js是目前最火的前端框架,是一套构建用户界面的框架，关注视图层。

框架： 是一套完整的解决方案，对项目的侵入性较大。
库(插件)：提供某一个小功能，对项目的侵入性较小，可以很容易的切换到其他库实现需求。

MVC 和 MVVM

MVC 是后端的开发概念

MVVM 是前端视图层的概念

前端视图层的分层开发思想，主要把每一个页面，分成了

M,V,VM

其中VM是MVVM的思想核心: VM是M和V之间的调度者

定义全局指令

在vue中指令，在调用时都以v-开头

使用Vue.directive() 定义全局的指令 v-focus

参数1是指令的名称，注意：在定义时不需要加v- 前缀
但是在调用时 必须 在指令名称前加上 v- 前缀
参数2是一个对象，在这个对象上，有一些指令相关的函数，这些函数可以在特定的阶段，执行相关的操作。

钩子函数的参数

钩子函数被赋予以下参数：

el: 指令所绑定的元素，可以用来直接操作DOM
binding: 一个对象，包含以下属性

name: 指令名，不包括v- 前缀
value: 指令的绑定值 v-my-directive=”1 + 1”, value的值是2.
expression: 绑定值的字符串形式， v-my-directive=”1 + 1”， expression 的值时”1 + 1”

vue get post jsonp

除了 vue-resource 之外，还可以使用 ‘axios’ 实现数据的请求

jsonp 原理

由于浏览器的安全限制，不允许AJAX访问 协议不同，域名不同 端口不同的数据接口，

可以通过动态创建script标签的形式，把script标签的src属性，执行数据接口的地址，因为script标签不存在跨域限制

jsonp 只支持GET请求

模块化 组件化

模块化： 代码逻辑的角度进行划分 node更多的是模块化 方便代码分层开发，保证每一个功能模块的职能单一。
组件化： 从UI界面的角度进行划分 前端的组件化，方便UI组件的重用
